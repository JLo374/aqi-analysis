# 作業反思報告

## 學生心得與反思

### 一、資料完整性（Data Completeness）

這次作業讓我第一次真正接觸到「髒資料」的處理。一開始看到避難收容所有將近 6000 筆資料，心想應該很完整吧，結果跑完檢查才發現有 86 筆座標跑到台灣邊界外面去了。雖然只占 1.4%，但如果不處理掉，後面的距離計算就會出現一些很離譜的數字。

我猜測這些問題可能是：有些是人工輸入時打錯數字（可能把經緯度對調了或少打一個小數點），有些可能是座標系統搞混了（TWD97 跟 WGS84 的數值差很多），還有一些可能是資料太舊沒更新。

老實說，一開始我其實沒想那麼多，就想說直接拿來用應該沒問題。但是 AI 生成的程式碼裡面有檢查座標範圍的機制，跑出來才發現有異常。這讓我意識到，不能完全相信原始資料，尤其是政府開放資料雖然方便但不見得每筆都正確。之後處理地理資料應該都要先做基本的驗證，像是檢查座標範圍、有沒有 (0,0) 這種明顯錯誤的點、座標系統是否一致等等。

不過這也讓我有點困惑，那些被過濾掉的 86 筆會不會其實是金門馬祖這些離島的資料？可惜我的程式設定的台灣邊界範圍可能有點保守。如果真的要做得更嚴謹，應該要去確認那些被移除的點實際上是什麼，而不是一刀切全部刪掉。但時間有限就先這樣處理了。

### 二、AI 協作與幻覺問題（AI Hallucination）

用 AI 寫程式確實快很多，但也踩了不少坑。

好的地方是，像是 Haversine 距離公式這種標準演算法，AI 生出來的程式碼基本上可以直接用，省了我很多查資料的時間。而且它可以一次整合多個需求，像是同時處理座標轉換、距離計算、風險評估，這如果要我自己從頭寫可能要花好幾個小時。

但是 AI 也會出錯。最明顯的例子是它一開始假設 MOENV API 的回應格式會有一個叫 `records` 的欄位，結果實際執行時發現 API 是直接回傳陣列，程式就報錯了。後來又發生一次，在計算風險等級統計時，它用了 `~df['is_indoor']` 這個語法，但因為欄位裡可能有 None 值就又出錯。這讓我意識到 AI 有時候會做一些假設，而這些假設不一定符合實際情況。

還有一個問題是，我不太確定 AI 用的套件版本是不是最新的，雖然這次沒遇到相容性問題，但感覺是個潛在風險。

我現在的做法是不會把 AI 生成的程式碼一次全部貼上去執行，而是一段一段測試。如果有報錯，我會先去看中間輸出的資料格式是什麼，然後才決定要怎麼修改。老師說的「AI 是工具不是老師」這句話還蠻有道理的，要保持懷疑的態度，不能太相信它。

### 三、空間推理與風險評估（Spatial Reasoning）

這部分是我覺得最有挑戰也最有趣的。一開始想說就單純算個距離，把每個避難所對應到最近的測站就好。但實際做的時候發現，風險評估其實沒那麼簡單。

首先是「距離近不代表風險高」這個概念。平均來說避難所離最近的測站只有 7.75 公里，看起來好像都蠻近的。但如果那個測站 AQI 只有 30，而且避難所是室內的，那根本沒什麼風險。反而是那些室外的避難所，就算測站 AQI 只有 60、70，在空污稍微嚴重的時候可能也不太適合。

情境模擬那段還蠻好玩的，因為現在空氣品質大部分都還可以（最高才 90），所以程式自動把高雄的某個測站設定成 AQI 150。這樣一來就能看到附近的避難所被標成「High Risk」，地圖上會出現紅色的點。這種模擬在實務上應該很有用，比如說演習的時候可以假設某個區域發生工業污染，看看周圍的避難所受影響的情況。

Folium 地圖真的幫助很大。本來只是一堆數字，放到地圖上之後就能很直觀地看出高風險區在哪裡、室內外避難所的分布是否平均。不過我發現地圖上的圖示有點多，5000 多個避難所全部標上去，放大看的時候會有點亂。如果要實際應用可能要加個篩選功能，或者依照縣市分圖層。

另外這次的風險評估規則其實蠻簡化的，只考慮了 AQI 跟室內外。真實狀況應該還要考慮容量、交通可達性、是否適合弱勢族群等等。但就這次作業來說，我覺得已經展現了「整合多維度資訊來做空間決策」的概念。

做完之後我在想，這種分析如果用在其他災害應該也適用，比如說淹水的時候可以整合淹水潛勢圖、避難所的樓層高度等等。空間分析確實不只是畫地圖而已，而是要把不同來源的資料兜在一起，找出有意義的模式。

---

## 總結

這次作業花的時間比預期的長（主要是在除錯），但學到的東西也比想像中多。除了技術層面的 Python、API、座標系統處理之外，更重要的是對「資料品質」有了新的認識，也學會了怎麼跟 AI 協作但不完全依賴它。

有點可惜的是時間不夠沒辦法做更細緻的分析，比如說我很想知道那 86 筆異常座標到底是什麼，或者把風險評估的規則調整得更完善。不過整體來說，對於空間分析的流程（從資料清理、計算、到視覺化）算是有了比較完整的體驗。
